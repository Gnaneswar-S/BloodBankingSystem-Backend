# Blood Banking System – Backend

RESTful backend service for managing blood donors, blood inventory and blood requests using Java and Spring Boot.

## Tech Stack
Java • Spring Boot • Maven • (H2/MySQL/PostgreSQL)

## Core Features
- Donor registration and management
- Search donors by blood group and city
- Blood inventory management
- Blood request creation and fulfillment
- Basic admin statistics

## Run Locally
./mvnw spring-boot:run

Application runs at: http://localhost:8080
